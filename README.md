# 🐍 Snake Game

A complete, polished Snake game built with **pure HTML, CSS, and vanilla JavaScript** — no frameworks, no external libraries, no build tools required.

![Snake Game](https://img.shields.io/badge/JavaScript-Vanilla-yellow) ![No Dependencies](https://img.shields.io/badge/Dependencies-None-brightgreen) ![GitHub Pages Ready](https://img.shields.io/badge/GitHub%20Pages-Ready-blue)

## 🎮 Features

- **Classic Snake gameplay** with smooth canvas-based rendering
- **Multiple control schemes**: Arrow keys, WASD, on-screen touch buttons, and swipe gestures
- **Three difficulty levels**: Easy, Medium, Hard (each changes game speed)
- **Score tracking** with persistent **High Score** saved via `localStorage`
- **Sound effects** (eating, turning, game over) generated with the Web Audio API — no audio files needed
- **Mute/Unmute** toggle with saved preference
- **Start / Pause / Resume / Restart** controls
- **Game Over screen** with final score and "Play Again" option
- **Fully responsive** design — works on desktop, tablet, and mobile
- **Modern UI** with smooth animations and a clean dark theme
- Prevents the snake from reversing directly into itself
